# La Apuesta de Pascal

App web interactiva que explora la Apuesta de Pascal como experimento mental.

La experiencia guía al usuario por varias etapas:

1. Introducción al argumento original.
2. Supuestos filosóficos que lo sostienen.
3. Elección personal (`Creo`, `No creo`, `No estoy seguro`).
4. Ruleta metafísica que selecciona un escenario posible.
5. Resultado reflexivo en forma de pregunta, no de veredicto.
6. Cierre final + sección `SABER MÁS` con explicación ampliada.

Objetivo: invitar a pensar si una decisión puede ser racional cuando el escenario del universo podría ser más complejo de lo que asumimos.
